# Getting started



## 1. Prerequisites

- Python version >=3.10

## 2. Installation
```bash
pip install git+https://github.com/OpenPecha/toolkit-v2.git
```

## 3. Configuration

1. _Setup and Development initialization_
```bash
pip install -U pip
pip install -e .
pip install .[dev]
pre-commit install

```

2. _Verify that all test cases pass_
```bash
pytest
```
